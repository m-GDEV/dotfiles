This is a simple list of all the files in my personal bin folder. 
It will be updated semi-regularly.
I will basically just explain what each bin entry does.

Author: Musa Ahmed
Date: July 3rd, 2021

--
aioz - unlocks aioz encrypted file that contains private key etc.

calc - a simple python-made calculator.

dotfiles - backup dotfiles.

ex - extract compressed archive, supports multiple formats.

fd - find command simplified for ease. uses certain defaults.

mc - make script. allows user to make a bash or python script and then makes it executable

opensc - opens latest screenshot taken

timer - simpler timer

trash - moves file to trash

ytm - YT_PLAYLIST project i made
